# Generate Resep - Fullstack Project

Proyek ini adalah aplikasi pembuat resep makanan yang menggabungkan frontend Android dan backend Django.

## Struktur Repositori
- `frontend-android/`: Kode sumber aplikasi Android (Kotlin/Compose).
- `backend-django/`: Kode sumber API backend (Python/Django).

## Workflow Tim (5 Orang)
Untuk kolaborasi tim, kami menggunakan alur kerja berikut:
1. **`main`**: Branch stabil. Hanya berisi kode yang sudah siap rilis.
2. **`develop`**: Branch integrasi. Semua fitur baru digabungkan di sini terlebih dahulu.
3. **Feature Branches**: Setiap anggota tim harus membuat branch baru dari `develop` untuk mengerjakan fitur spesifik.
   - Contoh: `git checkout -b feature/nama-fitur`

## Cara Berkontribusi
1. Clone repositori ini.
2. Pindah ke branch `develop`: `git checkout develop`
3. Buat branch fitur: `git checkout -b feature/tugas-saya`
4. Commit dan push branch fitur Anda.
5. Buat Pull Request (PR) ke branch `develop`.
